##  Getting Started

> **Note:** This is still under active development.

### 1. Clone the repository

```bash
git clone https://github.com/manavolt-dev/nuroo-app.git
cd nuroo-app
```


### 2. Install dependencies

```bash
npm install
```

### 3. Start development server
```bash
npx expo start
```
